# ULTRASURF
# CONTROLS
Move right: Right arrow / D
Move left: Left arrow / A
Shoot: Space
# CONTROLES
Mover para a direita: Seta direita / D
Mover para a esquerda: Seta esquerda / A
Atirar: Espaço
# ORB INFO
Blood orb: Although seeming dangerous, these orbs, when obtained, allows you to have a second life. 
When getting hit, you do not die and the orb is consumed.
Revolver orb: Yellow in color, these orbs give you a revolver that you can shoot bullets from for 10 seconds.
Soul orb: Cyan in color, these orbs make you invincible for 10 seconds. These however, are extremely rare.
# INFO DE ORBE
Orbe de sangue: Apesar de parecerem perigosas, essas orbes, quando obtidas, te permitem ter uma segunda vida.
Quando atingido, você não morre e a orbe é consumida
Orbe de revolver: Com cor amarela, essas orbes te dão um revolver que pode usar para atirar balas por 10 segundos
Orbe de alma: Com cor ciana, essas orbes te deixam invencíveis por 10 segundos. No entanto, elas são extremamente raras
